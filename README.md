# Portfolio

This repository hosts my portfolio webpage code. It uses HTML, CSS, SASS and JavaScript.

## Installation

Install dependencies:

* git
* make
* sass

Run these commands:

    git clone https://github.com/dtokayev/portfolio.git
    cd portfolio
    make

## Running

Open `index.html` using any browser.
